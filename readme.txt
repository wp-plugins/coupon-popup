== Description ==

Coupon Popup is a WordPress plugin from Sumoshopper that allows you to easily create customizable coupon widgets using shortcode syntax. 

== Installation ==

To install the plugin manually:

1. Download and unzip the plugin
1. Upload the `coupon-popup` directory to `/wp-content/plugins`
1. Activate the plugin through the **Plugins** menu in WordPress

== Frequently Asked Questions ==

**Are there different themes?**
    Yes, we have support for different themes. However, there's only one theme at the moment but we do plan on adding more in the future.
